⚡ Bulk Generation — Ek Saath Bohot Saari Rows

Neeche box mein har line ek barcode — format: Design, Color, Size, Qty
Comma , ya Tab se columns alag karo. Google Sheet se seedha copy-paste bhi chalega (Tab-separated). Qty na ho to 1 maan lenge.
Agar Design ya Color blank chhoda (merged cell jaisa), to upar wali row se inherit ho jayega.

📦 Bulk Set Input — WhatsApp se Seedha Paste (Flexible)

Jaisa WhatsApp pe order aata hai waisa hi paste karo — comma ya dash, dono chalenge, multiple designs ek saath, aur yeh 3 tareeke mix bhi kar sakte ho:

① Multi-line (range): 14320

② Ek-line, sizes ki list (dash ya comma): 5284K-BLUE-3XL,4XL,5XL,6XL,4 EACH → design=5284K, color=BLUE, 3XL/4XL/5XL/6XL = 4-4 each.
